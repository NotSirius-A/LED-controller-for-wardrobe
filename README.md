# LED-szafka-dlataty

This device is meant for controlling LED strips mounted on the inside of a wardrobe. The Controller will power LED strips according to which door was opened in order to light up the interior. This device is meant for long-term and  low-maintenance usage. It is highly configurable in software and is capable of some on-site hardware adjustments. The Device is equipped with error compensation functionality and lots of user convenience features. It is capable of powering up to 6 LED strip each drawing max 3A at max 24V while detecting the position of 6 different doors.
